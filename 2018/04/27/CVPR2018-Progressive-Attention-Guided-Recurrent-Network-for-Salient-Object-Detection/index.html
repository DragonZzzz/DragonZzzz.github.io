(CVPR2018)Progressive Attention Guided Recurrent Network for Salient Object Detection

大连理工的Huchuan Lu组在CVPR18上有多篇关于Salient object detection的文章，这是其中的一篇。

Motivation

  • 作者观察过去的FCN-based 方法中直接结合multi-level的特征而没有对其做其他处理（比如权重），比如不是所有特征，所有spatial position在salient detection中是一样重要的，，需要给feature的不同channel和不同的spatial position利用Attention机制加上权重，否则之后产生一些冗余的细节从而影响网络的效果。本文提出一个 attention guided network，progressively融合multi-level的特征。Attention机制可以更好的消除背景信息的干扰。
  • 同时，作者发现过去的方法会利用side-output features，但是浅层的特征没有global information，这会影响效果。利用一个multi-path recurrent connections，将global information从top layer 传递到shallow layers中，refine整个网络。

总的来说，作者在本文中使用了以下几个技巧或模块

  • Spatial Attention(SA)
  • Channel-wise Attention
  • Progressive Attention Guided
  • Multi-path Recurrent Connections

Spatial Attention and Channel-wise Attention

两种Attention，使用的都是Softmax Attention，将Attention作用在了不同的地方。
以Spatial Attention为例:
给定一个feature $f \in R^{W \times H \times C}$.首先生成一个Summarized feature map，
$m = W{s} * f + b{s}$
其中$*$代表卷积操作，之后利用Softmax操作产生在位置$l$上的attention权重，
$a{s}(l) = \frac{e^m(l)}{sum{l’ \in L}e^{m(l’)}}$
$a_{s}$是Attention map，
Channel-wise Attention同理。

Multi-Path Recurrent Guidance Module

因为缺乏global的信息，所有side-output通常含有冗余的细节。要解决这个问题，关键的一点是如何提高网络主干的学习效果。借鉴之前的工作，这里提出了这种方法。 将最后一层输出链接到之前的几层中去再训练，从而使得浅层的特征获得high-level的信息，拥有更好的识别能力。

定义$H_{l}$为第$l$个block的输出特征。论文中有这样的定义：

说明$t$时间$l$层的特征不仅和$t$时间$l-1$层的特征有关，还与$t-1$时刻最后一层的特征有关。
其中$U{l}^{t}$和$b{U_{l}}^{t}$为在训练中自适应学习的参数。

通过这种方法，可以使用较浅的网络结构依然能有效的提取到global、high-level的信息。相比于深层的网络结构，这种的开销更小更省时。
最后时刻的输出就是最终的预测saliency map。
